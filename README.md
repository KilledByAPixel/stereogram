# Stereogram Generator

Generate stereograms from depth map images in the browser.

## Features

- Load depth maps from presets or upload your own images
- Drag and drop support
- Adjustable depth intensity, repeat size, and texture scale
- Procedural noise-based pattern generation
- Pixelate mode
- Save generated stereograms as PNG
- Progressive rendering with live preview

## Usage

Open `index.html` in a browser. Load a grayscale depth map image where white = near and black = far. Adjust the sliders to tweak the stereogram output.

To view the hidden 3D image, relax your eyes and look "through" the screen (parallel viewing method).
